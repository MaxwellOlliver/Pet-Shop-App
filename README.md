### Get Started

Para começar digite os comandos abaixo no seu terminal com NPM:
  npm install && npm start
  
Para compilar a aplicação usando o electron-packager digite os seguintes scripts:

Windows:
  ```npm run package-win```
  
MAC:
  ``````npm run package-mac```
